Task API Documentation

Overview

This documentation covers the Task API endpoints which allow for managing tasks.

Endpoints

Get Task List

Returns a list of all tasks, each with an ID.

GET /tasks/

Example Response

[
{
    "id": 1,
    "description": "Buy groceries",
    "completed": false
},
{
    "id": 2,
    "description": "Read a book",
    "completed": true
}
]

Create a new task

POST /tasks/ 
{
    "description": "New Task",
    "completed": false
}

Example Response

{
    "id": 3,
    "description": "New Task",
    "completed": false
}

Delete A Task

DELETE /tasks/<task_id>/

Deletes a specific task by id.

Example Request

DELETE /tasks/3/

Example Response

Status: 204 No Content

Mark Task Complete/Incomplete

POST /tasks/<task_id>/toggle/

Toggles the completed status of a specific task by id.

Example Request

POST /tasks/3/toggle/

Example Response

{
    "id": 3,
    "description": "New Task",
    "completed": true
}
